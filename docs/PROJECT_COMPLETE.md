## 核心功能

### 1. 需求全生命周期管理
- 需求创建 → 评估 → BU决策 → 设计 → 确认 → 开发 → 测试 → 上线 → 交付 → 验收
- 自动状态流转
- 需求编号自动生成（REQ + 时间戳 + 序号）

### 2. 设计工作管理
- 三种设计类型：原型设计、UI设计、技术方案设计
- 设计工作记录跟踪
- 设计完成自动流转

### 3. 任务和工时管理
- 任务分配和状态管理
- 工时记录自动汇总到任务
- 支持按多维度筛选查询

### 4. 事项管理
- 事项类型：问题/风险/变更
- 严重程度：低/中/高/紧急
- 关联需求或任务

### 5. 数据统计
- 需求统计：各状态需求数、工时汇总
- 任务统计：各状态任务数、工时完成率
- 支持按项目、业务线、负责人筛选

## 关键技术决策

1. **状态流转自动化：** 各个环节完成后自动流转到下一状态
2. **工时自动汇总：** 工时记录自动汇总到任务实际工时
3. **需求编号生成：** REQ + 时间戳 + 序号，保证唯一性
4. **设计完成判断：** 三种设计全部完成后自动流转
5. **数据统计实时计算：** 基于实时数据统计，无需定时任务
