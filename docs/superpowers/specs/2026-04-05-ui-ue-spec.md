## 17. 交互规范

### 单双击行为
- **单击**：打开抽屉详情
- **双击**：打开独立页面（新窗口）
- 实现方式：250ms 延迟区分单击和双击

### 滚动行为
- 筛选栏吸顶（`position: sticky`）
- 吸顶后压缩为紧凑标签行
- 表格头部 `sticky`

### 动画
- 过渡时长：`0.15s` - `0.3s`
- 缓动：`ease`
- 弹窗滑入：从右滑入 `0.3s`
- 列表项 hover：轻微上移 + 阴影加深

## 20. Anti-Patterns（禁止使用）

基于 ui-ux-pro-max skill 的反模式规范：

| 规则 | 禁止 | 正确 |
|------|------|------|
| 图标 | 使用 Emoji | 使用 SVG 图标（Heroicons/Lucide） |
| 指针 | 点击元素无 cursor:pointer | 所有可点击元素添加 cursor:pointer |
| Hover | Hover 改变布局（scale） | 使用颜色/阴影变化 |
| 对比度 | 正文使用灰色-400 | 浅色模式下正文至少 gray-600 |
| 过渡 | 无过渡或过渡 > 500ms | 150-300ms 过渡 |
| Focus | 无 Focus 状态 | 键盘导航需可见 Focus 状态 |

## 21. Pre-Delivery 检查清单

交付前必须验证：

### 视觉质量
- [ ] 无 Emoji 作为图标（使用 SVG）
- [ ] 图标来自统一图标集（Heroicons/Lucide）
- [ ] Hover 状态不导致布局位移
- [ ] 使用主题色而非 var() 包装

### 交互
- [ ] 可点击元素有 cursor:pointer
- [ ] Hover 有明确的视觉反馈
- [ ] 过渡平滑（150-300ms）

### 布局
- [ ] 固定元素有适当边距
- [ ] 内容不被固定导航遮挡
- [ ] 响应式：375px、768px、1024px、1440px
- [ ] 移动端无水平滚动

### 可访问性
- [ ] 图片有 alt 文字
- [ ] 表单输入有 label
- [ ] 颜色不是唯一指示器
- [ ] 尊重 prefers-reduced-motion
